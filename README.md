# Memberships Management API

It's a REST API that manages memberships. It has two types of users: receptionists and administrators. It controls who makes sales, registers memberships, etc. Memberships can have time restrictions. It is possible to send notifications to members, notifying them that their membership ends in a few days, and it is also possible to send them general notices. It was designed for a gym, but can also be used by personal trainers and anyone who needs to manage client memberships.

## Deployment

To deploy this project run

```bash
cd Memberships-API
npm install
npm start
```
